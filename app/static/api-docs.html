Authentication

All API requests require a Bearer token or API key in the Authorization header.

// Option 1: JWT Token (from /api/v1/auth/login)
Authorization: Bearer eyJhbGciOiJIUzI1NiIs...

// Option 2: API Key (from dashboard)
Authorization: Bearer pgk_abc123...

Get a JWT Token

POST /api/v1/auth/login
Content-Type: application/json

{
  "email": "you@example.com",
  "password": "your-password"
}

// Response
{
  "user": { "id": 1, "email": "you@example.com", "plan_type": "pro" },
  "token": "eyJhbGciOiJIUzI1NiIs..."
}

Analyze Text

POST /api/v1/analyze-agent

Run the full multi-agent plagiarism detection pipeline on text.

POST /api/v1/analyze-agent
Authorization: Bearer <token>
Content-Type: application/json

{
  "text": "Your text to check for plagiarism...",
  "use_gpt_ai_detection": true,
  "excluded_domains": ["example.com"],
  "language": "en"
}

// Response
{
  "document_id": "abc123",
  "plagiarism_score": 12.5,
  "confidence_score": 0.87,
  "risk_level": "LOW",
  "detected_sources": [...],
  "flagged_passages": [...],
  "ai_detection": { "is_ai_generated": false, "ai_probability": 0.15 }
}

Analyze File

POST /api/v1/analyze

Upload a file (PDF, DOCX, TXT, TEX, PPTX) for plagiarism analysis.

POST /api/v1/analyze
Authorization: Bearer <token>
Content-Type: multipart/form-data

file: <your-file.pdf>
use_gpt_ai_detection: true

// Supported formats: PDF, DOCX, TXT, TEX, PPTX

Quick Check

POST /api/v1/quick-check

Fast plagiarism check using web search only (no AI detection).

{ "text": "Quick check this text..." }

// Returns: plagiarism_score, matched sources

AI Detection

POST /tools/ai-detect

Detect AI-generated text with per-model attribution (ChatGPT, Claude, Gemini, Copilot).

{ "text": "Check if this was written by AI...", "use_gpt": true }

// Response includes model_attribution:
{
  "ai_probability": 0.92,
  "is_ai_generated": true,
  "model_attribution": {
    "ChatGPT": 0.45, "Claude": 0.30,
    "Gemini": 0.15, "Copilot": 0.10
  }
}

Webhooks

Get notified when scans complete. Register a webhook URL and we'll POST results to it.

POST /api/v1/webhooks 
{
  "url": "https://your-app.com/webhook",
  "events": ["scan.complete"]
}

// Webhook payload sent to your URL:
{
  "event": "scan.complete",
  "document_id": "abc123",
  "plagiarism_score": 15.2,
  "risk_level": "LOW",
  "timestamp": "2024-01-15T10:30:00Z"
}
